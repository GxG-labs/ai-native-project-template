# Workbench

Temporary working area for the current task.

Use `input/` for files manually dropped into the project.

Use `output/` for intermediate generated results.

Promote durable files into the correct project folder when the task is complete.

Relationship to `briefs/`:

```text
briefs/          = what needs to be done
workbench/input/ = files to work with
workbench/output/ = intermediate results
```
