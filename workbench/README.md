# Workbench

Temporary drop zone for files provided for the current task.

Use `input/` for files the user drops in without specifying a destination.

Promote files to their correct home when the task is complete:
- Sources and notes → `context/sources/`
- Brand materials → `context/brand/`
- Draft outputs → `output/drafts/`
- Final outputs → `output/final/`

Intermediate generated results go directly to `output/drafts/`, not here.
