# Project AI System

This folder is the shared AI operating layer for the project.

Read order:

1. `constitution.md`
2. `project-context.md`
3. `instructions/general.md`
4. `instructions/organization.md` when creating or moving files
5. Relevant instructions, playbooks, skills, workflows, and checklists as needed
6. `../PROJECT_STRUCTURE.md` when deciding where files belong

`ai/` contains instructions for how AI assistants should work. It does not contain raw sources, final deliverables, datasets, or stable system documentation.
